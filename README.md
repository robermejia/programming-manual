## ✨ Features

- **Modern Tech Stack**: Built with React 18 and bundled with Vite for lightning-fast performance.
- **Syntax Highlighting**: Beautiful and accurate code block formatting powered by [Shiki](https://shiki.style/).
- **Smooth Navigation**: Client-side routing implemented using `react-router-dom` for a seamless user experience.
- **Clean UI Elements**: Scalable vector icons provided by `lucide-react`.
- **Responsive Design**: Easily accessible across different devices.

## 🛠️ Technologies Used

- **Framework**: React.js
- **Build Tool**: Vite
- **Routing**: React Router
- **Icons**: Lucide React
- **Syntax Highlighter**: Shiki

## 💻 Running Locally

To run this project on your local machine, follow these steps:

### Prerequisites

- Node.js installed on your machine.
- `npm` (Node Package Manager).

### Installation

1. **Clone the repository** (if applicable):
   ```bash
   git clone <your-repository-url>
   cd programming-manual
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser**:
   Navigate to the local URL provided in your terminal (usually `http://localhost:5173/`).

## 📦 Build for Production

To create a production-ready build:

```bash
npm run build
```

This will generate a `dist` folder with the compiled assets. You can preview the production build locally using:

```bash
npm run preview
```
